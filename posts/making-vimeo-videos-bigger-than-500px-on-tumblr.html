Making Vimeo and YouTube Videos Bigger Than 500px on Tumblr

A friend, @robalan, was searching everywhere on how to modify Tumblr's themes so that videos could be larger than 500px which is an arbitrary max size they set. There were a few scripts out there that were floating around but didn't work and it also looked like Tumblr was blocking jQuery or something because I we couldn't get jQuery working at all on the site. CSS might have worked, but the issue was it didn't scale to 16:9 dynamically.

In the end, I ended up with this little snippet and it works like a charm:

/**
 * Version 1.1
 * Makes Vimeo and YouTube videos on Tumblr larger than the max of 500px that Tumblr forces
 * Change the 650 to whatever you want, but do NOT add px or %
 * The height, by default, is set to auto which will calculate the 16:9 video size.
 * You can set the height to a static number as well if you want.
 * If you have any questions feel free to ask me: @oscargodson | oscargodson.com
 */
(function(){
  var iframes = document.body.getElementsByTagName('iframe')
  ,   newVimeoWidth = 650
  ,   newVimeoHeight = 'auto';
  for(x in iframes){
    if(iframes[x].src && iframes[x].src.indexOf('player.vimeo.com') > -1 ||
       iframes[x].src && iframes[x].src.indexOf('youtube.com') > -1){
      if(newVimeoHeight == 'auto'){
        newVimeoHeight = ( newVimeoWidth * 9 ) / 16;
      }
      iframes[x].width = newVimeoWidth;
      iframes[x].height = newVimeoHeight
    }
  }
})();
